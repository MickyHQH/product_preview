# QR Code Manager
A new app for scanning and generating QR code.

## Technology:
- Language: Kotlin (Jetpack Compose), DI (Hilt) <br/>
- Skills: Camera hardware + Google MLKit


## Implemented features:
- Scan QR Code
- Generate QR Code
- Share QR as image
- Save history, favorites
